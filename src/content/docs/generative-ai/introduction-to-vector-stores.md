title: Introduction to Vector Stores
description: How vector stores power modern AI applications like semantic search and RAG.
---

Vector stores (databases) are specialized databases designed to store and efficiently query high-dimensional vector embeddings.

## Key Features

- **Semantic Search:** Find documents that are semantically similar to a query.
- **Efficient Retrieval:** Fast lookup of top-K similar items from millions of documents.
- **RAG Support:** The backbone of Retrieval-Augmented Generation (RAG) applications.

## Popular Vector Stores

- **Pinecone:** Serverless vector database.
- **Chroma:** Open-source vector database.
- **Qdrant:** High-performance vector database.
- **Azure Cosmos DB:** Support for vector search.
