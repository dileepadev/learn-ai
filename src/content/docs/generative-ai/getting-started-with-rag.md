---

Retrieval-augmented generation (RAG) is a technique for enhancing the output of a large language model (LLM) by providing it with additional, relevant information from an external source.

## How RAG Works

1. **Retrieve:** Find relevant documents or snippets from a knowledge base.
2. **Augment:** Add the retrieved information to the user's original prompt.
3. **Generate:** Ask the LLM to provide an answer based on the augmented prompt.

## Benefits of RAG

- **Reduces Hallucinations:** Grounding the model in real data improves accuracy.
- **Current Information:** Provides the model with access to data beyond its training set.
- **Traceability:** It is easier to verify the source of the generated information.
