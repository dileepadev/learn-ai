## Tech Stack

- [Astro](https://astro.build/) with [Starlight](https://starlight.astro.build/) documentation theme
- [Tailwind CSS](https://tailwindcss.com/)
- Markdown & MDX content

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:4321` in your browser.
