Azure Cosmos DB is an ideal database for AI applications due to its global distribution, low latency, and vector search capabilities.

## Key Features

- **Vector Search**: Store and query high-dimensional vectors for RAG patterns.
- **Global Distribution**: Serve users worldwide with low-latency access.
- **Elastic Scale**: Scale throughput and storage independently.
